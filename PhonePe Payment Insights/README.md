# 📊 PhonePe Payment Insights Dashboard

An interactive and dynamic Power BI dashboard developed to analyze PhonePe payment transactions, user behavior, and service performance. The dashboard provides meaningful business insights using interactive visualizations, DAX measures, custom tooltips, and dynamic filtering.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------
## 📌 Project Overview
The PhonePe Payment Insights Dashboard is designed to monitor digital payment performance and help users explore transaction data through an intuitive and interactive interface. It enables users to analyze payment trends, transaction values, customer demographics, and service usage while generating dynamic business insights for better decision-making.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------
## 🎯 Objectives
- Monitor overall payment performance.
- Analyze monthly transaction trends.
- Track payment status performance.
- Identify high-performing services.
- Understand customer demographics.
- Compare weekday and weekend transaction patterns.
- Generate dynamic business insights.

## 💡 Dynamic Insights
The dashboard features an automated Insights panel powered by DAX measures.
Based on the selected **Month** and **Payment Status**, the insights update dynamically to display:
- Highest performing service
- Most successful transaction category
- Lowest performing transaction category
This enables quick business understanding without manual analysis.

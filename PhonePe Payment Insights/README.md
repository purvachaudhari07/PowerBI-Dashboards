# 📊 PhonePe Payment Insights Dashboard

An interactive Power BI dashboard developed to analyze PhonePe payment transactions, user behavior, and service performance. The dashboard provides meaningful business insights through dynamic visualizations, DAX measures, custom tooltips, and interactive filtering.

## 📌 Project Overview

The PhonePe Payment Insights Dashboard helps analyze digital payment transactions through an intuitive and interactive interface. It enables users to monitor transaction performance, identify trends, compare payment status, understand customer demographics, and generate business insights for better decision-making.

---

## 🎯 Objectives

- Monitor overall payment performance.
- Analyze monthly transaction trends.
- Track payment status performance.
- Identify top-performing services.
- Understand customer demographics.
- Compare weekday and weekend transaction patterns.
- Generate dynamic business insights.

## 💡 Dynamic Insights

The dashboard includes a dynamic Insights panel powered by DAX measures. Based on the selected Month and Payment Status, the insights automatically update to highlight:

- Highest performing service
- Most successful transaction category
- Lowest performing transaction category

This helps users quickly understand key business observations without manual analysis.
